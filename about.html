📽️ О проекте

MovieCatalog — учебный проект, разработанный в рамках курсовой работы по дисциплине «Информационные технологии и программирование».

🎯 Цель проекта

Создание современного, адаптивного веб-интерфейса для интернет-кинотеатра с возможностью поиска, фильтрации и сохранения фильмов в избранное.

🛠️ Используемые технологии

HTML5 CSS3 (Flexbox/Grid) JavaScript ES6+ localStorage Raleway (шрифт) Git / GitHub

✨ Реализованный функционал

  • ✓ Отображение каталога фильмов (12+ карточек)
  • ✓ Поиск по названию
  • ✓ Фильтрация по жанру
  • ✓ Сортировка по рейтингу
  • ✓ Добавление/удаление из избранного
  • ✓ Сохранение избранного в localStorage
  • ✓ Страница «Избранное»
  • ✓ Модальное окно с деталями фильма
  • ✓ Адаптивная вёрстка
  • ✓ Светлая и тёмная тема
