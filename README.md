# 🎬 MovieCatalog — Интернет-кинотеатр

## 📌 О проекте

**MovieCatalog** — это учебный веб-интерфейс интернет-кинотеатра, разработанный в рамках курсового проекта по дисциплине «Информационные технологии и программирование». Проект представляет собой современный, адаптивный и интерактивный сайт для просмотра каталога фильмов, их поиска, фильтрации, сортировки и сохранения в избранное.

🔗 **Демо:** [https://Dexon366.github.io/movie-catalog](https://dexon366.github.io/movie-catalog/)

---

## ✨ Реализованный функционал

| № | Функция | Описание |
|---|---------|----------|
| 1 | 📋 Каталог фильмов | Отображение 20 фильмов в виде карточек с постером, названием, годом, жанром и рейтингом |
| 2 | 🔍 Поиск по названию | Динамический поиск фильмов без перезагрузки страницы |
| 3 | 🎭 Фильтрация по жанру | Выбор жанра с анимированным подчёркиванием |
| 4 | 📊 Сортировка по рейтингу | Автоматическая сортировка от большего к меньшему |
| 5 | 💾 Избранное | Добавление/удаление фильмов с сохранением в localStorage |
| 6 | 🖼️ Модальное окно | Детальная информация о фильме: описание, режиссёр, продолжительность, трейлер |
| 7 | 🌙 Светлая/тёмная тема | Переключение темы с сохранением выбора |
| 8 | 📱 Адаптивность | Корректное отображение на ПК, планшетах и смартфонах |
| 9 | ⭐ Страница избранного | Отдельная страница со всеми сохранёнными фильмами |
| 10 | 🎬 Трейлеры | Кнопка «Смотреть трейлер» с переходом на YouTube |

## 🛠️ Технологический стек

| Технология | Назначение |
|------------|------------|
| **HTML5** | Семантическая структура страниц |
| **CSS3** | Стилизация, адаптивность (Flexbox/Grid), CSS-переменные для тем |
| **JavaScript (ES6+)** | Динамическая отрисовка, фильтрация, работа с localStorage |
| **LocalStorage** | Сохранение избранного и выбранной темы |
| **Git / GitHub** | Контроль версий, публикация проекта |
| **Raleway** | Основной шрифт интерфейса |

## 📁 Структура проекта
```
movie-catalog/
│
├── index.html # Главная страница (каталог)
├── favorites.html # Страница избранного
├── about.html # Страница «О проекте»
│
├── css/
│ ├── fonts.css # Подключение шрифтов
│ ├── global.css # Сброс стилей
│ └── style.css # Основные стили (светлая/тёмная темы)
│
├── fonts/ # Файлы шрифта Raleway (.woff/.woff2)
│
├── js/
│ ├── data.js # Массив из 20 фильмов
│ ├── favorites.js # Работа с избранным и localStorage
│ ├── modal.js # Модальное окно с деталями
│ ├── filters.js # Поиск, фильтрация, анимация полоски
│ ├── render.js # Отрисовка карточек фильмов
│ ├── favorites-page.js # Страница избранного
│ ├── theme.js # Переключение темы
│ └── main.js # Инициализация
│
├── img/ # Постеры фильмов
│
└── README.md # Документация проекта
```

## 🚀 Запуск проекта

### Локальный запуск

1. Клонируй репозиторий:
    ```bash
    git clone https://github.com/Dexon366/movie-catalog.git
    ```

2. Открой папку проекта:
    ```bash
    cd movie-catalog
    ```

3. Запусти файл `index.html` в любом современном браузере
